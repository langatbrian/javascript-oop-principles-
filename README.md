# JavaScript OOP Principles Practice

This project demonstrates Object-Oriented Programming (OOP) principles in JavaScript, focusing on writing clean, maintainable, and scalable code.

Built as part of The Odin Project JavaScript curriculum.

## Features

- Single Responsibility Principle
- Loosely Coupled Modules
- Separation of Logic and UI
- Clean Project Structure
- Scalable Code Design

## Concepts Covered

- Object-Oriented Programming (OOP)
- SOLID Principles
- Single Responsibility Principle (SRP)
- Loose Coupling
- Composition vs Inheritance
- Modular JavaScript

## Purpose

The purpose of this project is to practice writing maintainable and scalable JavaScript code using modern OOP principles.

## What I Learned

- How to separate application logic from UI
- How to design loosely coupled modules
- How to apply Single Responsibility Principle
- How to structure professional JavaScript projects
